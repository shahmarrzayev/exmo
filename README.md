Exmo Project
===============

## Description

Back-end service for Exmo.

## Installation

```bash
$ npm install
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Database migrations
To create a migration:
```bash
npx typeorm migration:create -n ExampleMigrationName
```

To migrate:
```bash
npm run typeorm-migrate
```
